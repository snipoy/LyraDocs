# Privacy Policy

Last updated: 28-03-2026

## 1. Overview

Lyra is a multipurpose Discord bot built with privacy in mind.

We do not store personal data and we only process the minimum data required for the bot to function.

## 2. Data Collection

Lyra does not maintain permanent storage of user data.

The following data may be processed temporarily in memory:

• Discord Server IDs  
• Discord User IDs  
• Discord Channel IDs  
• Message content (only when required to execute a command)  

This data exists only during command execution and is not saved to any database.

## 3. How Data Is Used

Data is used only to:

• Execute commands  
• Provide requested features  
• Send responses back to Discord  

Once the operation is complete, the data is discarded.

## 4. Data Storage

• No persistent databases are used for storing user or server data  
• No logs containing personal data are stored  
• No tracking or profiling is performed  

## 5. Data Sharing

We do not sell, share, or transfer any data to third parties.

## 6. AI and Model Training

Lyra does not use your data to train machine learning or AI models.

## 7. Data Retention

• Data is kept in memory only for the duration of processing  
• After processing, data is immediately discarded  

## 8. User Rights

You can stop using Lyra at any time by removing it from your server.

Since no data is stored, there is no retained data to request or delete.

## 9. Compliance

Lyra follows Discord’s Developer Terms of Service and Privacy requirements.

## 10. Changes to this Policy

We may update this policy when needed.

Continued use of Lyra means you accept the updated version.
