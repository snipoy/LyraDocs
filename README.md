# LyraBot

This repository contains the official legal documents for LyraBot, a multipurpose Discord bot.

## Contents

• Terms of Service  
• Privacy Policy  

## Purpose

These documents explain how Lyra operates, how data is handled, and what users can expect when using the bot.

They are used for:

• Transparency for users and server owners  
• Compliance with Discord Developer policies  

## Summary

Lyra is built with a privacy-first approach:

• No persistent storage of user data  
• Data is processed in-memory only during command execution  
• No data is sold, shared, or used for tracking  
• No data is used to train AI models  

## Updates

These documents may be updated over time to reflect changes in the bot or policies.
